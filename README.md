## 🚀 Highlighted Projects

### 🎣 [SustainCast Fisheries Web Application](https://github.com/tharindperera/SustainCast-Fisheries-Web-App)
> **Stack:** React, Vite, Tailwind CSS, FastAPI, Python | *Individual Contribution*
- Built an end-to-end full-stack web application to operationalize predictive machine learning models, helping fishers forecast trip profitability before departure.
- Developed an interactive dashboard with live KPIs, cross-filtering geographic charts, and a custom simulation tool to model fuel price fluctuations.
- Optimized and deployed the backend CatBoost regression model via FastAPI.

### ⚖️ [SMOTE Oversampling Web App](https://github.com/tharindperera/SMOTE-Oversampling-Web-App)
> **Stack:** JavaScript, Python | *Individual Contribution*
- Developed an educational full-stack visualizer and simulator designed to help students, researchers, and machine learning practitioners intuitively understand class imbalance.
- Implemented visual and geometric interactive simulations of popular oversampling and cleaning algorithms to demonstrate the mathematical mechanics of data balancing.

### 🏢 [Remote Worker Productivity Prediction] 
> **Stack:** Python, CatBoost, FAMD, K-Means | *Group Project*
- Addressed the challenge of objectively measuring remote employee efficiency by developing a predictive modeling modeling framework.
- Applied **Factor Analysis of Mixed Data (FAMD)** alongside **K-Means clustering** to handle mixed-type datasets and high multicollinearity.
- Deployed an optimized CatBoost regression model (Test $R^2$ of `0.9991`, Test RMSE of `0.055`) in a web application to provide real-time productivity scoring.
- Utilized SHAP values to explain and identify daily work habits and calendar discipline as primary drivers of productivity.
